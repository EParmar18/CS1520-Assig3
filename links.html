Braille Buddy





In order to assist blind/non-blind people with learning braille, our team created a machine a which takes any string input on the Arduino serial monitor and converts that to braille, so you can learn letters and words. Our program converts the letters and punctuation to braille and then controls 6 six servo motors that create bumps on the surface to simulate braille. We use the following tools and languages
  • C++
  • Adruino
  • Servo Motors
  • React Native
